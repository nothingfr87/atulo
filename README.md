# Audio Player

Audio Player is a C program made to play audio files with SDL2 Library

---

## Requirements:

- GCC to build
- SDL2

--- 

## Supported OS:
- Windows (Not Tested)
- Linux
- MacOS (Not Tested)

---

## Features:

- Works from terminal directly
- Supports only `wav` for now
- lightweight
